# Gridiron Engine

Football game simulation engine with statistical modeling and realistic game flow.

## Overview

This engine provides the core simulation logic for football games, including:
- State machine-based game flow (19 game states)
- Probability-driven play outcomes
- Player skill calculations
- Penalty system
- Injury tracking
- Configurable game options
